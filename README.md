# 🚩 CTF 학습 노트

## 💡 이 사이트는 무엇인가요?

마치 요리 레시피북처럼, 해킹 기법을 체계적으로 정리한 **개인 학습 노트**입니다. 각 "레시피"(기법)는 실전에서 바로 사용할 수 있도록 정리되어 있으며, 왜 이 방법이 효과적인지도 함께 설명합니다.

### ✨ 주요 특징

📋 **실전 중심의 치트시트**
> 대회 중 긴박한 순간에도 빠르게 찾아서 복사-붙여넣기 할 수 있는 명령어와 페이로드 모음

🧠 **원리부터 이해하기**
> 단순히 "이렇게 하세요"가 아닌, "왜 이것이 작동하는가?"를 함께 탐구합니다

🎯 **게임처럼 레벨업**
> Easy(초보자) → Medium(중급) → Hard(고급) → Expert(전문가) → Insane(천재?) 단계로 점진적 성장

🔗 **지식의 네트워크**
> XSS를 배우다가 관련된 JavaScript 트릭을 발견하고, 그것이 또 다른 문제로 이어지는 학습의 여정

{% hint style="info" %}
**살아있는 문서** 🌱

이 사이트는 CTF 대회에서 새로운 것을 배울 때마다 업데이트됩니다. 마치 정원을 가꾸듯이 지식을 축적하고 있습니다.
{% endhint %}

## 🎯 왜 이 사이트를 만들었나요?

### 📖 한국어로 배우는 해킹
영어로 된 훌륭한 자료들을 한국어로 번역하고, 우리 문화와 환경에 맞게 다시 정리합니다. 마치 좋은 외국 요리책을 우리 입맛에 맞게 번안하듯이요.

### 🧭 나만의 학습 로그
"어라, 이거 저번에 풀어봤는데..." 하고 기억은 나는데 정확한 방법이 떠오르지 않던 경험, 다들 있으시죠? 이 사이트는 그런 순간을 위한 **미래의 나를 위한 친절한 메모장**입니다.

### 🎮 레벨업 시스템
게임처럼 단계적으로 성장할 수 있도록:
- 🟢 **Easy**: "아하, 이렇게 하는구나!" (첫 성공의 기쁨)
- 🟡 **Medium**: "음... 생각을 좀 해봐야겠는데?" (두뇌 풀가동)
- 🟠 **Hard**: "이건 좀 어렵네... 힌트가 필요할지도" (진짜 실력 테스트)
- 🔴 **Expert**: "여러 기법을 조합해야 하는구나" (마스터 레벨)
- ⚫ **Insane**: "...이게 가능해?" (천재들의 놀이터)

### 🕸️ 지식의 거미줄
하나의 취약점을 공부하다 보면 자연스럽게 관련된 다른 기법들로 연결됩니다. SQL Injection을 배우다가 Union 공격을 알게 되고, 그것이 다시 Time-based blind SQLi로 이어지는 식이죠.

## 🗂️ 어떻게 구성되어 있나요?

현재는 원본 구조를 유지하고 있지만, 점차 **카테고리별 → 난이도별** 구조로 재구성될 예정입니다:

```
🌐 Web          → 웹사이트를 해킹하는 다양한 방법들
🔧 Binary Exp   → 프로그램의 메모리를 조작하는 기술
🔐 Crypto       → 암호를 풀거나 만드는 수학적 마법
🔍 Forensics    → 숨겨진 정보를 찾아내는 탐정 놀이
⚙️ Reversing    → 프로그램의 비밀을 역추적하기
📱 Mobile       → 모바일 앱의 보안 허점 찾기
☁️ Cloud        → 클라우드 환경의 특수한 취약점들
```
