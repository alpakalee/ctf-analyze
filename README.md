# 🚩 CTF 학습 노트

📋 **빠른 참조**: 자주 쓰는 명령어와 스크립트를 복사해서 바로 사용할 수 있습니다

🧠 **깊이 있는 이해**: 공격 기법의 작동 원리와 이유를 상세히 설명합니다

🔗 **관련 문제 연결**: 비슷한 기법을 사용하는 문제들을 크로스 링크로 연결합니다

## 🎯 이 프로젝트의 목적

### 1. 한국어 번역 및 재구성
원본 영문 콘텐츠를 한국어로 번역하고, 한국 CTF 환경과 커뮤니티에 맞게 재구성합니다.

### 2. 개인 학습 기록
CTF 대회에서 만난 문제들의 풀이 과정과 사고 과정을 상세히 기록합니다.

### 3. 실력 향상을 위한 체계적 정리
- 난이도별로 정리하여 점진적인 학습 경로 제공
- 관련 문제들을 연결하여 기법별 깊이 있는 학습 가능
- 자주 사용하는 페이로드와 스크립트를 빠르게 참조

### 4. 지식 축적 및 공유
배운 내용을 체계적으로 정리하여 나중에 쉽게 찾아볼 수 있도록 하고, 필요시 다른 사람들과 공유합니다.

## 🗂️ 사이트 구조

이 사이트는 **카테고리별 → 난이도별** 구조로 재구성되었습니다:

```
📁 Web (웹 해킹)
📁 Binary Exploitation (바이너리 익스플로잇)
📁 Cryptography (암호학)
📁 Forensics (포렌식)
📁 Reverse Engineering (리버싱)
📁 Mobile (모바일)
📁 Cloud (클라우드)
📁 Networking (네트워킹)
📁 Languages (프로그래밍 언어)
📁 Linux / Windows (운영체제)
📁 Other (기타)
```

각 카테고리 내에서 난이도별로 분류됩니다:
- **Easy**: CTF 입문자를 위한 기초 문제
- **Medium**: 기본 개념을 이해한 후 풀 수 있는 문제
- **Hard**: 심화 기법이 필요한 문제
- **Expert**: 복합적인 취약점 체이닝이 필요한 고급 문제
- **Insane**: 창의적인 접근이 필요한 최고난이도 문제

## 🚀 시작하기

왼쪽 사이드바에서 원하는 주제를 선택하거나, 오른쪽 상단의 검색 기능을 사용하세요!

{% hint style="success" %}
**추천 학습 순서**
1. 관심 있는 카테고리의 Easy 문제부터 시작
2. 관련 문제 링크를 따라가며 비슷한 기법 학습
3. 실제 CTF에서 만난 문제를 직접 풀어보며 연습
4. 새로운 기법을 배울 때마다 이 사이트에 정리
{% endhint %}
